# site/

Pure-static interactive viewer (S2, not yet implemented).

## Planned structure

```
site/
  index.html          entry point
  js/                 vanilla JS SPA
  css/
  data/
    coverage.json     generated by scripts/coverage.py
    decls.json        symlink or copy from extracted/
    corpus.json       flattened corpus built from corpus/**/*.yaml
```

## Pipeline

```
extracted/decls.json (or names-fallback.json)
    + corpus/**/*.yaml
        → scripts/coverage.py → site/data/coverage.json
        → site build script   → site/data/corpus.json
```

The site requires no server — all data is bundled as static JSON.
